# Test ReactJS/Electron application

### Install

1) Clone project from remote repository
 
    $ git clone https://github.com/midsummer/electron-react-feed.git

2) Install necessary packages with `npm`

    $ npm install

3) Run renderer server

    $ nmp run server

4) Start electron

    $ nmp start
